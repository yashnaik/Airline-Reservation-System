Airline Reservation App 

Completed By : Yash Naik
Video : AirlineReservationApp.mp4 (Please look at the demo)

Working:

This App has been made using MySql in Visual Studio using C#. This App is a UI based App.

Forms:

Form1: First Form accepting and authenticating Users either Admin or Guest

Form2: Admin related form . Displaying all the essentials to add Flight to the Database

Form 3 (User booking form): Guest related Form which displays all the flights in the Database

Form 4 (Seating): Displays which seats are available for the particular flight

Form 5 (Confirmation): This displays the confirmation page to book the Ticket
